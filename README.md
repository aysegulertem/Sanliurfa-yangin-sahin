# 🔥 Şanlıurfa Yangın Şahin

Akıllı yangın algılama ve erken uyarı sistemi.

## Proje Amacı

Ormanlık ve kırsal alanlarda meydana gelebilecek yangınları erken aşamada tespit ederek müdahale süresini azaltmayı amaçlamaktadır.

## Kullanılan Teknolojiler

- Python
- Flutter
- Veri Analizi
- Sensör Verileri
- Harita Sistemleri
- Bildirim Sistemleri

## Özellikler

- Duman analizi
- Sıcaklık takibi
- Nem ölçümü
- Risk skorlama
- Konum bazlı uyarılar
- Harita desteği
